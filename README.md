\# Customer Churn Prediction App



A Machine Learning web application that predicts customer churn using a Random Forest Classifier and Streamlit.



\## Dataset



\* Records: 7,043 customers

\* Churn Rate: 26.54%



\## Features Used



\* Tenure Months

\* Monthly Charges

\* Total Charges

\* Churn Score

\* CLTV



\## Model



\* Random Forest Classifier

\* Hyperparameter Tuning using RandomizedSearchCV

\* Accuracy: 84.85%



\## Application Features



\* Customer Churn Prediction

\* Churn Probability Score

\* Risk Level Indicator

\* Interactive Streamlit Dashboard



\## Technologies



\* Python

\* Pandas

\* NumPy

\* Scikit-Learn

\* Streamlit

\* Joblib
